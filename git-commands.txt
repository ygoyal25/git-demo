git init ---> Initialize a Folder with git version control
git add <File Name> ---> Add a file for git to track
git add . ----> Add all the files for git to track
git commit -m <Commit Message> ----> Commit / Save the changes



--------------------------------
git log ----> Shows history of the commits

git checkout <commit hash> ----> Goes back and forth in the history to check the data present at that point of time
 
git checkout <branch name> ---> Goes to the latest changes or the branch


---------------------------------
git remote add origin <repo link> ---> Adds a remote to the local git repo (creates a link between your local and remote repo)
git push origin master ---> Pushes the code to github or remote repo

git clone <repo link> ----> copies the remote repo to local system


Resolving conflicts
---------------------------------
git checkout <target branch> ---> checkout to branch where you want to merge your code
git pull ---> pull the latest changes
git checkout <source branch> --->  checkout to your source branch which you want to merge
git merge <target-branch> ---> merge the target branch
